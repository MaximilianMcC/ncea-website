## Wireframes
This is the first wireframe version. The client liked the overall design of it, however they wanted to have some more content.
![First wireframe](https://raw.githubusercontent.com/MaximilianMcC/ncea-website/main/other/wireframe%201.png)

After the feedback I made another version of the wireframe where I have added all of the content that they wanted.
![Second wirefrme](https://raw.githubusercontent.com/MaximilianMcC/ncea-website/main/other/Wireframe%202.png)
